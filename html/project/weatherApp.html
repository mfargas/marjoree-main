React Weather App

Key Takeaways

This Weather App is a step up from a few other projects where I experimented with location services. I am incorporating the OpenWeatherAPI with Google's GeoCoding API. The reason I use both API calls is due to a challenge I faced wherein OpenWeatherAPI's free tier offers location services based on coordinates. Given that users are more familiar with addresses, I want to meet this challenge with a secondary GeoCode API provided by Google, allowing me to translate the address given into latitude & longitude positions. This provides information the OpenWeatherAPI call can use to generate the weather data.
